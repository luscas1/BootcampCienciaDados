# Guia Rápido — Como Usar Markdown no VS Code

## Criar um arquivo Markdown
Basta criar um arquivo com extensão `.md`, como:
notes.md

## Abrir o Preview
- **Ctrl + Shift + V** – abre o preview lado a lado  
- **Ctrl + K V** – abre o preview em outra aba  

---

## Sintaxe Básica
---
# Títulos

# Título 1

## Título 2

### Título 3
---

### Negrito e Itálico
**negrito**

*itálico*


### Listas
- item

- item

- item


### Código Inline e Blocos de Código
exemplo inline
`print("Olá, mundo!")`

em bloco
```python
print("Hello")
```

### Quebra de Linha
Adicionar dois espaços no final da linha  
ou deixar uma linha em branco.

---

## Ferramentas Úteis no VS Code
- **Markdown All in One** – atalhos, tabelas, TOC  
- **Todo Tree** – lista de TODOs  
- **TODO Highlight** – destaca notas no código  

---

## Dicas
- Use Markdown para notas, documentação, tarefas e resumos.  
- Crie uma pasta `docs/` para organizar.  
- O preview atualiza automaticamente ao salvar.
