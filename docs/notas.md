# Resumo — História e Características do Python

## Origem
Python foi criado em 1989 por **Guido van Rossum**, inspirado na linguagem **ABC**.  
Inicialmente desenvolvido como um passatempo, tornou-se uma linguagem completa, limpa e acessível.

---

## Mantras da Linguagem
- Simples e intuitiva  
- Open source  
- Código legível como inglês  
- Adequada para tarefas do dia a dia  

---

## Linha do Tempo
- **1991** – Primeira versão pública  
- **1994** – Python 1.0  
- **2001** – Fundação da **PSF (Python Software Foundation)**  
- **2008** – Lançamento do **Python 3**, com quebras de compatibilidade  

---

## Versatilidade
- Tipagem **forte** e **dinâmica**  
- Multiplataforma e multiparadigma  
- Comunidade ampla e ativa  
- Curva de aprendizado baixa  

## OBS

Sempre utilizar snake case
Nomes sugestivos as variáveis, funções etc
